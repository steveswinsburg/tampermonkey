# tampermonkey

A collection of tampermonkey scripts. 

**printable-aldi-recipe-layout.js**

Adds a button that activates on the ALDI recipe site to alter the layout for better printing on a single page. The CSS that ALDI supply still includes a bunch of the navigation and other stuff that makes printing harder.
Also, these are the best recipes! https://www.aldi.com.au/recipes/

**jira-badges.js**

A comprehensive script for enhancing Jira. All of the rules can be enabled or disabled as you choose.

Rules:
1. If ticket is a bug, improvement or support request, ensure there is work logged once the ticket is resolved
2. If ticket is a bug, show a badge if publication is not specified or it is publicise or 'publicise in detail' without a public description.
3. If ticket is a bug, improvement or story, show a badge if there is no fix version, but only if the ticket is completed and only if the resolution is Fixed or Done
4. If ticket is a bug, improvement or story and there is no component, show a badge.
5. If ticket has the blocked label, show a badge.
